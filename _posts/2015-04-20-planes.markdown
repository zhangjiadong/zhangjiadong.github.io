### java多线程

```java
public static int[] sort(int[] data) {
        if (0 == data.length) {
            return data;
        }
        //i、j分别为数组两头的指针
        int i = 0;
        int j = data.length - 1;
        while (i != j) {
            if (data[i] >= 0) {
                i++;
            } else {
                if (data[j] > 0) {
                    int temp = data[i];
                    data[i] = data[j];
                    data[j] = temp;
                    i++;
                } else {
                    j--;
                }
            }
        }
        return data;
    }
```
